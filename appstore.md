### Description (4000 chars max)

TinyCSV is a fast, native CSV and TSV editor for macOS. Open a file, see your data as a live table, and edit the raw text — all in one window. No spreadsheet overhead. No import wizards. Just clean, direct access to your data.

Features:
- Side-by-side editor and live table preview
- Automatic delimiter detection (comma, tab)
- Syntax highlighting with color-coded columns
- Sortable table with click-to-jump between editor and preview
- Folder sidebar with directory browsing
- Multi-tab editing for working with several files at once
- Quick open file finder (Cmd+P)
- Auto-save as you type
- Native Find & Replace
- Adjustable font size (Cmd+/Cmd-)
- Word wrap toggle (Option+Z)
- Line numbers toggle (Option+L)
- File watcher — reloads automatically when files change externally
- Handles quoted fields, escaped delimiters, and multiline values
- Light and dark mode — follows your system
- Opens .csv, .tsv, and .txt files directly from Finder

Built entirely with native macOS technologies. No Electron. No web views. Just SwiftUI and AppKit, fast and lightweight.
